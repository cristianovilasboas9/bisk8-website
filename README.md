# BISK8 — Landing Page

Landing page for BISK8, the AI-powered wardrobe app.

## Stack

- React 18
- Vite 6
- HighCruiser custom font

## Setup

```bash
npm install
npm run dev
```

## Structure

```
public/
  images/       # Logos, step screenshots, hero image
  fonts/        # HighCruiser font
  *.png         # Section mockups
src/
  App.jsx       # Main landing page component (9 languages)
  main.jsx      # React entry point
```

## Languages

FR, EN, DE, ES, IT, PT, AR, ZH, RU

## Deploy

```bash
npm run build
```

Output in `dist/` — deploy to Vercel, Netlify, or any static host.
